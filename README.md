## Deskripsi
**HelpMe!** adalah platform pemesanan jasa serbaguna berbasis web yang dirancang untuk menghubungkan pengguna yang membutuhkan bantuan dengan penyedia jasa dalam satu sistem terintegrasi. Platform ini mendukung berbagai jenis layanan, mulai dari kebutuhan harian hingga permintaan jasa khusus, dengan alur pemesanan yang sederhana dan terstruktur.

Aplikasi ini memungkinkan pengguna untuk membuat permintaan jasa, berkomunikasi langsung dengan penyedia jasa, serta memberikan penilaian terhadap layanan yang diterima. Di sisi lain, penyedia jasa dapat menerima dan mengelola pekerjaan melalui dashboard yang disediakan oleh sistem.

Proyek ini dikembangkan sebagai bagian dari tugas akademik dan berfokus pada pengembangan **Minimum Viable Product (MVP)** berbasis web. Repository ini berisi implementasi fitur inti aplikasi beserta dokumentasi pendukung.

---

## Fitur Utama
- Registrasi dan login pengguna  
- Pembuatan dan pengelolaan request jasa  
- Penerimaan job oleh penyedia jasa  
- Sistem chat antara pengguna dan penyedia jasa  
- Kategori jasa serta fitur pencarian dan filter  
- Sistem rating dan ulasan  
- Dashboard pengguna  

---

## Ruang Lingkup Aplikasi
Aplikasi difokuskan pada pengembangan platform web dengan fitur pemesanan jasa dan komunikasi dasar antara pengguna dan penyedia jasa.

**Tidak termasuk dalam cakupan saat ini:**
- Aplikasi mobile native (Android/iOS)  
- Sistem pembayaran otomatis  
- Verifikasi identitas tingkat lanjut
